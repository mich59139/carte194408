Aide à la navigation

Carte : Romanche & Oisans — Isère (été 1944). Cliquez sur les marqueurs pour ouvrir les fiches.

  • Zoom : molette, +/−, ou pincement sur mobile.
  • Déplacement : cliquez-glissez (ou glisser au doigt sur mobile).
  • Calques : utilisez le sélecteur en haut à droite (Août 1944 (focus) / Autres (contexte)).
  • Marqueurs groupés : cliquez sur les cercles numérotés pour déployer, puis cliquez un point pour sa fiche.
  • Raccourcis : touche Esc ferme cette aide. Bouton ⌂ pour revenir à la vue d'ensemble.
  • Exporter une image : utilisez la version “Carte + export PNG”.
